Stream API Examples

Explore the available API endpoints and learn how to integrate Stream Payments into your application.

Products

Manage your products and pricing.

GET /api/products

Example cURL Request:

curl http://localhost:3000/api/products?page=1&size=10
GET /api/products/:id

Example cURL Request:

curl http://localhost:3000/api/products/123
POST /api/products/create

Example cURL Request:

curl -X POST http://localhost:3000/api/products/create \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Premium Subscription",
    "price": 199.99,
    "currency": "SAR",
    "type": "ONE_OFF",
    "description": "Monthly premium plan"
  }'

Manage customer information and profiles.

GET /api/consumers

Example cURL Request:

curl http://localhost:3000/api/consumers?page=1&size=10
GET /api/consumers/:id

Example cURL Request:

curl http://localhost:3000/api/consumers/123
POST /api/consumers/create

Example cURL Request:

curl -X POST http://localhost:3000/api/consumers/create \
  -H "Content-Type: application/json" \
  -d '{
    "email": "customer@example.com",
    "name": "Ahmad Ali",
    "phone": "+966501234567"
  }'

Payment Links

Create and manage payment links for easy checkout.

GET /api/payment-links

Example cURL Request:

curl http://localhost:3000/api/payment-links?page=1&size=10
POST /api/payment-links/create

Example cURL Request:

curl -X POST http://localhost:3000/api/payment-links/create \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Test Payment",
    "amount": 99.99,
    "consumer": {
      "email": "customer@example.com",
      "name": "Ahmad Ali",
      "phone": "+966501234567"
    },
    "product": {
      "name": "Premium Plan",
      "price": 99.99
    }
  }'

Invoices

Create and manage invoices for your customers.

GET /api/invoices

Example cURL Request:

curl http://localhost:3000/api/invoices?page=1&size=10
POST /api/invoices/create

Example cURL Request:

curl -X POST http://localhost:3000/api/invoices/create \
  -H "Content-Type: application/json" \
  -d '{
    "consumerId": "123",
    "items": [
      {
        "productId": "456",
        "quantity": 1
      }
    ],
    "dueDate": "2024-12-31"
  }'
