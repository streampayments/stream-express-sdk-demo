Welcome to the Stream Express Boilerplate. Explore the examples below to get started with payment integration.

Shop (Product Listing) API Examples Health Check View Documentation

Available Routes:

  • GET / - Home page (you are here)
  • GET /shop - Product listing with checkout
  • GET /examples - API examples and documentation
  • GET /health - Server health check
  • GET /api/products - List all products (JSON)
  • GET /api/consumers - List all consumers (JSON)
  • GET /api/payment-links - List payment links (JSON)
  • GET /api/subscriptions - List subscriptions (JSON)
  • GET /api/invoices - List invoices (JSON)
  • GET /api/coupons - List coupons (JSON)
